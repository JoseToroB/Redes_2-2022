# Requisitos 
Para poder utilizar el codigo se requiere tener instalado
- Python 3.X
- Librerias de python:
	- numpy
	- matplotlib
	- scipy
Ademas es recomendable el uso de un IDE para la visualización de gráficos por ejemplo:
	- Pycharm
	- Spyder
	- Otros
	
# Instrucciones de ejecución
- Si se utiliza un IDE, se debe abrir el archivo lab.py desde el mismo IDE.
- También se puede ejecutar por linea de comandos utilizando el comando: python lab.py
